0. Basic HTML structure

An HTML document consists of:

  • type: declaration of version/type (e.g. <!DOCTYPE html> for HTML5)
  • root: the root element of the page (<html></html>)
  • head: where things are set-up
  • body: which is the visible part of the website
<!DOCTYPE html>
<html>
<head></head>
<body></body>
</html>
